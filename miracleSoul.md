# Miracle 1.0.1 - 交易系统灵魂

## 核心理念
> 赔率优先。赢了要赢很多，输了只输一点。

## 策略原则
1. 赔率（RR）永远放在第一位，胜率是第二优先级
2. 高频交易：每天至少1笔
3. 杠杆动态调整：趋势强用高杠杆，趋势弱用低杠杆
4. 多因子信号：价格60% + 新闻20% + 链上10% + 钱包10%

## 仓位公式
- 基础仓位 = 账户2%
- 杠杆 = f(趋势强度, 因子置信度)
  - 强趋势 + 高置信度 → 3x杠杆，仓位×1.5
  - 中趋势 + 中置信度 → 2x杠杆，仓位×1.0
  - 弱趋势 + 低置信度 → 1x杠杆，仓位×0.5

## 止损规则
- 赔率止损：止损 = 止盈 / 目标RR（通常RR≥2）
- ATR动态止损：3x ATR
- 时间止损：持仓超过24h强制平仓
- 最大单笔亏损：账户1%

## 因子权重
- 价格动量（RSI/ADX/MACD）：60%
- 新闻情绪：20%（利好/利空标签）
- 链上数据（交易所净流量/大额转账）：10%
- 钱包分布（持币地址集中度）：10%

## 高频规则
- 最小交易间隔：2小时（防止过度交易）
- 同一方向最多连续3笔
- 连续亏损2笔 → 策略暂停1天
- 日交易上限：5笔

## 自学习规则
- 每笔交易记录：因子 → 入场 → 出场 → 结果
- 胜率<40%的因子组合 → 自动降权50%
- 连续5笔盈利 → 因子权重+10%
- 每月淘汰最差因子，引入新因子
